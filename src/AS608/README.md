# 指纹识别模块

AS608模块可以用于指纹识别，下面将介绍一下AS608的基本使用。



> 推荐可以查找一些AS608的用户手册进行查阅，里面有更详细的开发信息，不过对Arduino的开发好像没有什么帮助...

## 快速上手

在Arduino IDE开发需要安装**Adafruit_Fingerprint**库

在示例中可以直接调用"录入指纹"、"删除指纹"、“识别指纹”三个模块，用户可以根据这三个例程和自己的需求去修改成需要的代码。
